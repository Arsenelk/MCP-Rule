# MCP 使用规则知识库

1. Normal MCP tasks: read `mcp_usage_rules.md` first.
2. Troubleshooting or incident append: read `mcp_issue_log.md` only when needed.
3. PowerShell script work: read `powershell_script_rules.md` only for `.ps1` creation, review, or edits.
4. Update/audit/expand this knowledge base: read `mcp_auto_update_protocol.md`, then relevant low-frequency files.
5. Preserve real disk path spellings exactly: `AssiatanceKnowledge` and `Knowdledge` are known on-disk names in existing paths; do not autocorrect them unless a verified replacement path exists.
